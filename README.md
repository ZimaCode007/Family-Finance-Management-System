# Family Finance Management System

A team-based financial management system for families.

## Tech Stack
- Backend: Spring Boot
- Frontend: Vue 3
- Database: MySQL

## Features
- Family member login
- Asset & transaction tracking
- Score-based decision system
- Recurring financial events

## Project Structure
- backend/
- frontend/
- docs/
